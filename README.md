# 🌿 VaryaCode

**Développeuse web indépendante en Dordogne** — Sites accessibles pour indépendants, artisans et associations.

## 💡 Le projet

VaryaCode, c'est une conviction : le web professionnel ne devrait pas être réservé aux grosses structures.

J'accompagne les **indépendants**, les **petites entreprises locales** et les **associations** qui veulent exister en ligne — sans jargon, sans usine à gaz, sans budget démesuré.

> Commencer simple · Construire solide · Évoluer intelligemment
